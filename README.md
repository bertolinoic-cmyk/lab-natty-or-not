# Chrono Trigger: O Filme (Trailer Conceito com IA)

## 🎥 Resumo do Projeto
Este projeto apresenta a ideia visual de um trailer cinematográfico (live-action) para um filme baseado no clássico jogo **Chrono Trigger**, explorando o poder das IAs Generativas para trazer o clássico pixelado ao realismo fotorrealista.

## 💻 Tecnologias Utilizadas
- **Microsoft Copilot / Bing Image Creator:** Utilizado para a geração das imagens realistas do trailer.
- **ChatGPT / Gemini:** Utilizado para criar o conceito das cenas e a descrição dos prompts de imagem.

## 🍿 Processo de Criação
1. **Roteirização:** Definimos as cenas marcantes que fariam parte do início do trailer (o Crono correndo para o portal e o clássico corte para o relógio de bolso marcando o tempo).
2. **Engenharia de Prompt:** Construímos comandos detalhados em inglês para extrair o máximo de realismo das imagens geradas por IA.
3. **Geração das Imagens:** Usamos as ferramentas para dar vida às cenas.
